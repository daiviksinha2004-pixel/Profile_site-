## ML & Analytics Portal — BFSI Predictive Platform
**Stack:** FastAPI, React, Llama 3, PostgreSQL, Random Forest, Recharts.
- Full-stack predictive portal with a Glassmorphism UI for debt-collection and life-insurance
  forecasting, serving real-time Random Forest and Logistic Regression inference.
- Llama 3-powered SQL Agent for natural-language data querying with interactive Recharts analytics,
  secured with JWT authentication and Alembic-managed schema migrations.

## Local-Hosted Desktop AI Agent
**Stack:** Python, Tkinter, Ollama (local LLM), Whisper (STT), Edge-TTS.
- A real-time, fully offline voice assistant — Whisper for speech-to-text, Edge-TTS for speech, and a
  local Ollama LLM — so all data stays private on-device.
- Async processing + multithreading for concurrent voice capture and LLM inference, wrapped in a
  Tkinter desktop UI.

## Financial Analysis Portal
**Stack:** FastAPI, React/Vite, SQL Agent, Recharts, PostgreSQL.
- Analytics dashboard with a natural-language SQL Agent bot and Recharts KPI visualizations over
  PostgreSQL.

## Airbnb Price Prediction
**Stack:** XGBoost, Python, Google Colab.
- Trained an XGBoost regression model for price forecasting, with extensive EDA and rigorous feature
  engineering.
