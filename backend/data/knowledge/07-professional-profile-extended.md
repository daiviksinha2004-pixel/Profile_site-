## Flagship Project — BFSI Analytics Platform

An enterprise-grade analytics platform for Banking, Financial Services, and Insurance (BFSI) operations.

**Objective:** Give business teams actionable insights, forecasting capabilities, customer analytics,
and AI-powered intelligence through a centralised platform.

**Tech Stack:**
- Frontend: React, Tailwind CSS, TypeScript
- Backend: FastAPI, Python (async)
- Database: PostgreSQL
- Security: JWT Authentication, AES-256-CBC Encryption, Role-Based Access Control
- AI: RAG (natural-language SQL agent), Predictive Analytics, Forecasting Models

**Key Features:**
- Multi-tenant architecture
- Secure authentication and payload encryption
- Interactive dashboards with 200+ chart instances
- KPI monitoring and S-curve forecasting
- Customer analytics and treatment dashboards
- Data ingestion pipelines (Oracle → PostgreSQL)
- In-app AI assistant (Llama 3 NL-to-SQL agent)
- Audit logging and role-based access
- 7-layer security model

**Scale:** 30,000+ policies/campaign month, 40 GB+ datasets, 7 prediction engines, 60+ API routes.
